a tag "container" é a tag que envolve os itens e é nela que aplicaremos as propriedades "display: flex" ela vai transformar os itens de dentro em "flex-itens"

propriedades relacionadas:

  1. display — define como um elemento será exibido (block, inline, flex, grid, etc.).
  2. flex-direction — define a direção dos itens no container flex eles são:
    • row (padrão): à direção do texto, esquerda para a direita
    • row reverse: é o contrario
    • column: ordenação de cima pra baixo, em coluna unica
    • column reverse: o contrario de baixo pra cima
  3. flex-wrap — controla se os itens devem quebrar para a linha de baixo quando não couberem.
  4. flex-flow — atalho que combina flex-direction e flex-wrap.
  5. justify-content — alinha os itens no eixo principal (horizontal por padrão).
  6. align-items — alinha os itens no eixo transversal (vertical por padrão).
  7. align-content — alinha várias linhas de itens no container (quando há quebra de linha).
  8. gap — define o espaçamento entre os itens no container flex.

propriedades relacionadas flex-itens

  1. flex-grow — define quanto um item pode crescer em relação aos outros dentro do container.
  2. flex-basis — define o tamanho inicial do item antes da distribuição de espaço.
  3. flex-shrink — define quanto um item pode encolher em relação aos outros.
  4. flex — atalho que combina flex-grow, flex-shrink e flex-basis.
  5. order — define a ordem de exibição dos itens dentro do container flex.
  6. align-self — permite sobrescrever o align-items para um item específico.
