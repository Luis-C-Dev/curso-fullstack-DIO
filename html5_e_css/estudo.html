Vou te montar um **resumão estilo dicionário** dos principais comandos de **centralização/alinhamento no CSS**, principalmente **Flexbox** e alinhamento de texto. --- ### **1. `text-align`** (alinhamento de texto dentro do container) | Comando | Descrição | | --------- | ---------------------------------------------------------------------------------- | | `left` | Alinha o texto à esquerda (padrão) | | `right` | Alinha o texto à direita | | `center` | Centraliza horizontalmente o texto | | `justify` | Justifica o texto, ajustando espaçamento entre palavras para ocupar toda a largura | --- ### **2. `justify-content`** (alinhamento de itens no **eixo principal** de um flex container) | Comando | Descrição | | --------------- | --------------------------------------------------------------- | | `flex-start` | Itens alinhados no início do container | | `flex-end` | Itens alinhados no final do container | | `center` | Itens centralizados no eixo principal | | `space-between` | Espaço igual entre itens, primeira à esquerda, última à direita | | `space-around` | Espaço igual ao redor de cada item | | `space-evenly` | Espaço igual **entre e nas extremidades** de todos os itens | --- ### **3. `align-items`** (alinhamento de itens no **eixo cruzado** de um flex container) | Comando | Descrição | | ------------ | ----------------------------------------------------------- | | `flex-start` | Itens alinhados no início do eixo cruzado | | `flex-end` | Itens alinhados no final do eixo cruzado | | `center` | Itens centralizados no eixo cruzado | | `baseline` | Itens alinhados pela linha de base do texto | | `stretch` | Itens esticam para ocupar toda a altura disponível (padrão) | --- ### **4. `align-self`** (sobrepõe `align-items` para **um item específico**) | Comando | Descrição | | ------------ | -------------------------------------------------- | | `auto` | Herda do `align-items` do container | | `flex-start` | Item alinhado no início do eixo cruzado | | `flex-end` | Item alinhado no final do eixo cruzado | | `center` | Item centralizado | | `baseline` | Alinhado pela linha de base | | `stretch` | Item esticado para ocupar toda a altura disponível | --- ### **5. `align-content`** (alinhamento de **linhas múltiplas** em flex container) | Comando | Descrição | | --------------- | ----------------------------------------------------- | | `flex-start` | Linhas agrupadas no início do container | | `flex-end` | Linhas agrupadas no final | | `center` | Linhas centralizadas | | `space-between` | Espaço igual entre linhas | | `space-around` | Espaço igual ao redor das linhas | | `stretch` | Linhas esticam para ocupar toda a altura do container | --- Se você quiser, posso te
